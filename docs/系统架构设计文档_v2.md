## 1. 项目概述

**项目名称**: 币钱袋 (CryptoQuant)
**项目类型**: 数字货币量化交易移动应用
**核心技术栈**:
- 后端: Python 3.11+ / FastAPI / SQLAlchemy (Async) / PostgreSQL / Redis
- 前端: Flutter 3.x / Dart / Riverpod / GoRouter
- 数据库: PostgreSQL + TimescaleDB (时序数据)
- 消息队列: Redis Streams

### 3.2 核心模块详解

#### 3.2.1 配置管理 (config.py)

```python
Settings(BaseSettings)
├── app_name: str              # 应用名称
├── app_version: str           # 版本号
├── debug: bool                # 调试模式
├── secret_key: str            # 加密密钥（必填，无默认值）
├── database_url: str           # 数据库连接（必填）
├── redis_url: str             # Redis 连接
├── jwt_secret_key: str        # JWT 密钥（必填）
├── jwt_algorithm: str         # HS256
├── access_token_expire_minutes: int  # 30分钟
├── refresh_token_expire_days: int     # 7天
└── cors_origins: list[str]    # CORS 白名单
```

**特性**:
- 环境变量 `.env` 加载
- 敏感信息无默认值，强制环境变量注入
- `@lru_cache` 单例模式

#### 3.2.2 数据库层 (database.py)

```
AsyncEngine (create_async_engine)
├── pool_size: 10
├── max_overflow: 20
├── pool_pre_ping: True
└── echo: debug 模式输出 SQL

async_sessionmaker
├── class_: AsyncSession
├── expire_on_commit: False
└── autocommit: False
```

**会话获取**:
- 依赖注入: `async def get_db() -> AsyncGenerator[AsyncSession]`
- 上下文管理器: `async with get_db_context() as session:`

#### 3.2.3 Redis 连接层 (redis.py)

```
ConnectionPool (redis.asyncio)
├── max_connections: 20
├── decode_responses: True
└── 全局单例 + asyncio.Lock 线程安全
```

**主要用途**:
- 行情数据缓存（TTL: 10秒）
- 会话管理
- 限流计数

#### 3.2.4 策略引擎 (strategy_engine.py)

```
BaseStrategy (抽象基类)
├── name: str
├── strategy_type: str
├── config: StrategyConfig
└── methods:
    ├── analyze(klines) -> Signal | None
    ├── calculate_pnl(entry, current, side)
    └── check_stop_loss(current, entry, side)

实现策略:
├── MAStrategy (双均线)
│   └── 金叉买入/死叉卖出
├── RSIStrategy (RSI 超买超卖)
│   └── RSI < 30 买入 / RSI > 70 卖出
├── BollingerStrategy (布林带)
│   └── 价格触及下轨买入/上轨卖出
└── GridStrategy (网格交易)
    └── 固定价格区间内低买高卖
```

**Signal 数据结构**:
```python
@dataclass
class Signal:
    action: str          # buy / sell / close
    confidence: float    # 0.0 - 1.0
    entry_price: Decimal
    stop_loss_price: Decimal
    take_profit_price: Decimal
    reason: str
    timestamp: datetime
```

#### 3.2.5 交易所适配器 (exchange_adapter.py)

```
BaseExchangeAdapter (ABC)
├── _shared_client: httpx.AsyncClient  # 类级别单例
└── abstract methods:
    ├── get_ticker(symbol) -> Ticker
    ├── get_klines(symbol, interval, limit) -> list[Kline]
    ├── get_orderbook(symbol, limit) -> OrderBook
    ├── get_balance() -> list[Balance]
    ├── get_positions(symbol) -> list[PositionInfo]
    ├── create_order(...) -> OrderResult
    ├── cancel_order(order_id, symbol) -> bool
    └── get_order(order_id, symbol) -> OrderResult

实现:
├── BinanceAdapter
│   └── BASE_URL = "https://api.binance.com"
└── OKXAdapter
    └── BASE_URL = "https://www.okx.com"
        └── _to_inst_id() 符号转换

工厂函数:
get_exchange_adapter("binance"|"okx", api_key, secret_key, passphrase)
```

#### 3.2.6 安全模块 (security.py)

```
密码处理:
└── CryptContext(schemes=["bcrypt"])
    ├── hash_password(password) -> str
    └── verify_password(plain, hashed) -> bool

JWT Token:
├── create_access_token(data, expires_delta)
│   └── exp + type:"access" + 30分钟默认
├── create_refresh_token(data, expires_delta)
│   └── exp + type:"refresh" + 7天默认
├── decode_token(token) -> dict
└── verify_token(token, token_type) -> dict

API Key 加密 (AES-256-Fernet):
├── _get_encryption_key()
│   └── sha256(secret_key) -> base64url
├── encrypt_api_key(plaintext) -> str
└── decrypt_api_key(ciphertext) -> str
```

## 7. 核心数据流

### 7.1 用户登录流程

```
┌────────┐                         ┌────────────┐                    ┌─────────┐
│ Client │                         │  FastAPI   │                    │ PostgreSQL │
└───┬────┘                         └─────┬──────┘                    └────┬────┘
    │                                   │                               │
    │  POST /api/v1/auth/login          │                               │
    │  {email, password}               │                               │
    │──────────────────────────────────►│                               │
    │                                   │                               │
    │                                   │  AuthService.authenticate()  │
    │                                   │──────────────────────────────►│
    │                                   │                               │
    │                                   │  SELECT * FROM users         │
    │                                   │  WHERE email = ?             │
    │                                   │◄──────────────────────────────│
    │                                   │                               │
    │                                   │  verify_password()           │
    │                                   │                               │
    │                                   │  create_access_token()      │
    │                                   │  create_refresh_token()      │
    │                                   │                               │
```
