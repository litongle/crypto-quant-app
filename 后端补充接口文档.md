## 📊 1. 资产汇总接口

**接口路径**：`GET /asset/summary`

**接口说明**：返回用户聚合后的总资产信息，包括总资产、累计盈亏、可用余额、冻结余额。

**请求参数**：无（从 Token 中获取 user_id）

**请求示例**：
```
GET /api/v1/asset/summary
Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
```

**响应参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| totalAsset | number | ✅ | 总资产（USDT） |
| totalPnL | number | ✅ | 累计盈亏（USDT） |
| totalPnLPercent | number | ✅ | 累计收益率（%，正数赚，负数亏） |
| availableBalance | number | ✅ | 可用余额（USDT） |
| lockedBalance | number | ✅ | 冻结余额（挂单占用，USDT） |
| todayPnL | number | ✅ | 今日盈亏（USDT） |
| todayPnLPercent | number | ✅ | 今日收益率（%） |
| updatedAt | string | ✅ | 数据更新时间（ISO 8601） |

**响应示例**：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "totalAsset": 125680.50,
    "totalPnL": 25680.50,
    "totalPnLPercent": 25.68,
    "availableBalance": 98500.25,
    "lockedBalance": 27180.25,
    "todayPnL": 1234.56,
    "todayPnLPercent": 0.99,
    "updatedAt": "2026-04-21T02:00:00Z"
  }
}
```

## 📈 2. 批量行情接口

**请求示例**：
```
GET /api/v1/market/tickers?symbols=BTC,ETH,SOL,BNB,DOGE

**请求示例**：
```
GET /api/v1/asset/positions
GET /api/v1/asset/positions?exchange=binance&side=long

## 📉 4. 权益曲线接口

**接口路径**：`GET /asset/equity-curve`

**接口说明**：获取账户权益曲线历史数据，用于 Dashboard 收益图表展示。

**请求参数**：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| days | number | ❌ | 30 | 查询天数（7/30/90/180/365） |
| exchange | string | ❌ | all | 交易所筛选 |

**请求示例**：
```
GET /api/v1/asset/equity-curve?days=30
GET /api/v1/asset/equity-curve?days=90&exchange=binance
```

**响应参数**：

| 字段 | 类型 | 必填 | 说明 |

## 🔄 移动端对接说明

### API Client 配置

移动端 `ApiClient` 已配置好基础设置，只需在 `ApiConfig` 中填入实际 Base URL：

```dart
class ApiConfig {
  static const String baseUrl = 'https://api.biqiandai.com/api/v1'; // TODO: 填入实际地址
  static const Duration timeout = Duration(seconds: 30);
}
```

### Provider 对接方式

当前移动端使用模拟数据，对接后端时只需修改 Provider 中的数据获取方式：

```dart
// 资产汇总
final assetProvider = FutureProvider<Asset>((ref) async {
  final response = await ApiClient.get('/asset/summary');
  return Asset.fromJson(response['data']);
});

// 行情列表
final tickersProvider = FutureProvider<List<MarketTicker>>((ref) async {
  final response = await ApiClient.get('/market/tickers');
  return (response['data'] as List)
      .map((e) => MarketTicker.fromJson(e))
      .toList();
});

// 持仓列表
final positionsProvider = FutureProvider<List<Position>>((ref) async {
  final response = await ApiClient.get('/asset/positions');
  return (response['data'] as List)
      .map((e) => Position.fromJson(e))
      .toList();
});

// 权益曲线
final equityCurveProvider = FutureProvider.family<EquityCurve, int>((ref, days) async {
  final response = await ApiClient.get('/asset/equity-curve', queryParams: {'days': days});
  return EquityCurve.fromJson(response['data']);
});
```

### 刷新策略

| 页面 | 刷新频率 | 实现方式 |
|------|----------|----------|
| Dashboard 资产 | 30秒 | `Timer.periodic` + `ref.invalidate` |
| 行情列表 | 10秒 | `Timer.periodic` + `ref.invalidate` |
| 持仓列表 | 30秒 | `Timer.periodic` + `ref.invalidate` |
| 权益曲线 | 5分钟 | 缓存优先，按需刷新 |
| 策略列表 | 手动刷新 | Pull-to-refresh |
