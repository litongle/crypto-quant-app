## 架构演进

### 策略引擎（6种策略 + 规则引擎 + 自动交易）

| 策略 | 类型 | 核心逻辑 |
|------|------|---------|
| 双均线 (MA) | 内置 | 短期/长期均线交叉 |
| RSI | 内置 | 超买超卖区间反转 |
| 布林带 (Bollinger) | 内置 | 上下轨突破回归 |
| 网格 (Grid) | 内置 | 价格区间等分挂单 |
| 马丁格尔 (Martingale) | 内置 | 亏损加倍+趋势判断+连亏保护 |
| 自定义规则 (Rule) | DSL | 14种指标 + AND/OR 逻辑 + 交叉检测 |

**运行器**：StrategyRunner 单例 → asyncio.Task → K线获取 → 策略分析 → 信号持久化 → 自动交易 → WS推送

### 交易所适配器（~1150行，3交易所）

| 特性 | 说明 |
|------|------|
| 异常体系 | RateLimitError / NetworkError / OrderRejectedError |
| 重试+限流 | 指数退避 1→2→4s，下单不重试，查询3次 |
| 安全 Decimal | `_safe_decimal()` / `_safe_divide()` |
| OKX 特殊处理 | Passphrase 警告 + 服务器时间同步 |
| Huobi 缓存 | accountId TTL 5分钟 + 失败清理 |
| 止损止盈 | `create_stop_order()` + 降级机制 |

### 设计系统 v3.1

| 特性 | 说明 |
|------|------|
| 字体 | Geist Sans（替代 Inter）+ JetBrains Mono |
| 主色 | Indigo #6366F1(暗) / #4F46E5(亮) |
| 流体缩放 | clamp() 字体/间距/网格，480px→1440px |
| 双主题 | data-theme 属性 + localStorage 持久化 |
| CSS 变量 | --cq- 前缀设计令牌体系 |
| 图标 | 全量 SVG（Lucide 风格），禁止 Emoji |
